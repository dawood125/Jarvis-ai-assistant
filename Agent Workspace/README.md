# Agent Workspace

Purpose:
- Keep project execution organized.
- Track decisions and avoid repeating mistakes.
- Preserve context across daily 1-hour sessions.

How to use:
1. Update plans/current-roadmap.md when scope or priorities change.
2. Update context/project-context.md when architecture or constraints change.
3. Update preferences/user-preferences.md and preferences/coding-style.md when you set new standards.
4. Update security/security-rules.md before adding risky system actions.
5. Update skills/build-skills.md when we establish reusable workflows.
6. Add a short note in logs/daily-session-log.md after each session.
