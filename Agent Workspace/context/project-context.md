# Project Context

## Goal
Build a personal JARVIS assistant for Windows that feels premium, modern, and futuristic while staying lightweight for current hardware.

## Constraints
- OS: Windows
- Hardware: 8GB RAM class target
- Budget: free-first
- Privacy: local-first data policy

## Architecture Direction
- Frontend: React + Vite + Tailwind
- AI: Groq primary with OpenRouter fallback
- Memory: persistent local storage from day one
- Safety: confirm destructive actions before execution

## Key UX Principle
- Chat is the primary interaction surface.
- Right panel and telemetry are support systems, not the main focus.
