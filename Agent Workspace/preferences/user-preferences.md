# User Preferences

## Learning Mode
- Build and learn together in each session.
- Explain what, why, and how after implementation.
- Keep progress practical and structured for growth.

## Product Preferences
- Premium futuristic visual quality is required.
- Avoid generic template-like UI.
- Keep data local in early phases.

## Delivery Preference
- Daily progress in small, complete chunks.
- End each session with recap and next step.
